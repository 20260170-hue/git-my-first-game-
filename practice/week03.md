Pygame을 사용해서 간단한 게임 프로그램을 제작
기본 창 생성부터 시작해서 점점 기능을 추가
최종적으로는 탄막 피하기 게임 형태까지 발전시킴
구현한 기능들
화면 생성 및 원(플레이어) 표시
방향키로 이동
Shift로 속도 증가
방향키 입력 시 색상 변경
화면 밖 이동 제한
네모 오브젝트(탄막) 생성
여러 방향에서 날아오는 탄막 구현
충돌 판정
게임오버 기능
시간 기반 점수 시스템
시간이 지날수록 난이도 증가
# AI와의 대화에서 배운 것
코드를 단계별로 기능을 추가하면서 게임을 점진적으로 개발하는 방법
오류가 발생했을 때:
에러 메시지를 보고 원인을 찾는 방법
NameError → 라이브러리 import 문제라는 것
키 입력 처리 방식:
get_pressed() → 계속 누르는 입력 처리
이전 상태와 비교 → "한 번만 실행" 구현 가능
충돌 판정 방법:
pygame.Rect를 이용하면 간단하게 처리 가능
게임 개발 기본 구조:
입력 → 처리 → 그리기 → 업데이트
# 코드들의 특징 정리
## 원 오브젝트의 움직임
방향키 입력에 따라 x, y 좌표가 변함
Shift 키를 누르면 속도가 2배 증가
실시간으로 좌표가 바뀌면서 자연스럽게 이동
## 색 변화
방향키를 눌렀을 때 색상이 변경됨
랜덤 RGB 값을 사용하여 다양한 색 생성
키를 계속 누르고 있을 때는 색이 계속 바뀌지 않도록 처리함
## 화면 밖으로 나가지 않음
원의 반지름을 고려해서 위치를 제한
화면 경계를 벗어나면 다시 안쪽으로 고정
플레이어가 항상 화면 안에 있도록 유지
## 탄막(오브젝트) 시스템
작은 네모 형태로 생성
리스트에 저장하여 여러 개 동시에 관리
위, 아래, 왼쪽, 오른쪽 모든 방향에서 등장
시간에 따라: 생성 확률 증가, 이동 속도 증가
## 충돌 처리
원과 네모를 사각형으로 변환하여 충돌 판정
충돌 시 게임오버 상태로 변경
점수 시스템
플레이한 시간을 기준으로 점수 증가
오래 버틸수록 점수가 높아짐

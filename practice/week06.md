# 6주차 실습 기록

## 사용한 에셋
- 이미지: (snake_graphics.zip, opengameart.org)
- 사운드: (파일명, 출처)-> 찾는 중

## 사용한 AI 프롬프트 (요약)
1.이미지 png 파일로 다운 후에 이미지 에셋 파일 코드에 어떻게 적용하는지 요청
2.이미지 파일을 집어 넣은 후 맵의 칸 수 확장 요청
3.뱀 이미지 에셋 파츠가 여러 개라 움직일 때 오류가 생겨 자연스럽게 방향을 전환할 수 있도록 수정 요청
4.속도로 인한 난이도 조정 및 클리어 조건, 장애물 추가 요청

## AI 답변에서 도움이 된 것
1.이미지 에셋 png 파일로 다운 받은 후 이 파일을 어떻게 적용해야 하는지 잘 답변해준 것.
2.이미지 에셋 파츠가 여러 개인데 이 부분을 코드로 어떻게 적용하고 나누어지는지 파트 별로 잘 정리해준 것.
## AI 답변을 수정하거나 버린 것
답변 중 다른 내용이 추가되었거나 기존에 있던 내용이 없어진 부분을 수정함

## 적용 결과
- 잘 된 것: 스네이크 이미지 에셋이 머리 상하좌우, 몸통 상하좌우 등 파츠가 여러 개 있어 에셋이 잘 들어가는 것과 자연스럽게 움직이는 것이 표현될 지 걱정스러웠지만 잘 적용됨. 이외에도 먹이 이미지 에셋 장애물 랜덤화, 조건부 클리어 등이 잘 적용됨.
- 어려웠던 것:AI가 코드를 만들어낼 때마다 기존에 있었던 걸 지우거나 이상한 걸 추가하는 경우가 많았음. 예를 들어 이미지 에셋은 안 들어가고 다른 규칙이 적용되거나, 이미지 에셋은 들어갔는데 자연스럽게 표현이 되지 못하거나 이미지 에셋 이외에 다른 기능이 전부 제거되는 등의 시행착오가 있었음. 
- 다음에 시도할 것: 게임의 새로운 조건이나 창의적인 부분이 부족하다고 판단하여 기획을 고치거나 스네이크 템플릿은 그대로 가지만, 방향성에 변화를 주는 방식을 시도할 예정. 사운드 에셋과 배경 에셋 등을 추가로 찾고 적용할 예정.
